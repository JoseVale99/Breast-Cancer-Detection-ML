COMPARACIÓN DE MODELOS DE APRENDIZAJE AUTOMÁTICO PARA LA PREDICCIÓN DE TIPO DE CÁNCER DE MAMA (BENIGNO O MALIGNO)
Planteamiento del problema
El cáncer de mama es un cáncer muy común a nivel mundial. Para ayudar a los médicos a clasificar el tipo de cáncer ya sea benigno o maligno se han utilizado técnicas de aprendizaje automático, para realizar esto el aprendizaje automático utiliza modelos, cada modelo tiene un porcentaje de precisión, una mayor precisión indica un mejor modelo para el aprendizaje automático.
Objetivo General
Construir un sitio web donde se representen diferentes modelos de clasificación para la detección de tipo de cáncer de mama(benigno y maligno).
Alcances
  • Leer un dataset para luego explorarlo en diferentes modelos
  • Obtener el puntaje de precisión de cada modelo.
  • Entregar la predicción para un cierto porcentaje del dataset que serán considerados los datos de prueba.
  • Comparar diferentes modelos de clasificación para el aprendizaje automático (Machine Learning) para la predicción de Cáncer de mama si es benigno o maligno.

¿Cómo se utiliza la App?

Solo tienes que importar el dataset de entrenamiento con la extensión (.csv). Posteriormente podrás navegar en la página donde encontraras los modelos de aprendizaje automático (ML) para la predicción de Cáncer de Mama.
