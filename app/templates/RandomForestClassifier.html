MODELO CLASIFICADOR DE ARBOLES ALEATORIOS
¿Qué es?

El bosque aleatorio o el bosque de decisiones aleatorias es un algoritmo de aprendizaje automático supervisado que se utiliza para la clasificación, la regresión y otras tareas mediante árboles de decisión. El clasificador de bosque aleatorio crea un conjunto de árboles de decisión a partir de un subconjunto seleccionado al azar del conjunto de entrenamiento. Básicamente es un conjunto de árboles de decisión (DT) de un subconjunto seleccionado al azar del conjunto de entrenamiento y luego recopila los votos de diferentes árboles de decisión para decidir la predicción final. En este algoritmo de clasificación, utilizaremos conjuntos de datos de flores de IRIS para entrenar y probar el modelo.

La clasificación aleatoria de bosques utiliza una técnica de conjunto para obtener el resultado deseado. Se entrenan varios árboles de decisión utilizando los datos de entrenamiento. Este conjunto de datos contiene observaciones y características que se elegirán al azar cuando se dividan los nodos. En un sistema de selva tropical se utilizan varios árboles de decisión.

Predicción de salida con los datos de prueba

Random Forest es un clasificador que contiene una serie de árboles de decisión en varios subconjuntos del conjunto de datos dado y toma el promedio para mejorar la precisión predictiva de ese conjunto de datos. Aquí, estamos utilizando el método RandomForestClassifier de la clase de conjunto para implementar el algoritmo de clasificación de bosque aleatorio.
